# BNU_OpenCS

欢迎访问 **BNU_OpenCS**！这是一个北师大计算机科学学习资源平台，旨在汇集并共享课程学习与考试相关的信息，帮助同学们更高效地学习和准备考试。

## 项目简介

**BNU_OpenCS** 旨在提供以下内容：

- **课程信息**：包括学习内容、考核方式。
- **课程评价**：同学们的学习体验分享与建议。
- **考试回忆**：期中期末考试的题型与重点整理。
- **课程笔记**：同学们贡献的详细学习资料。

该项目由北师大计算机专业学生共同维护，目标是建立一个长期稳定的开放学习资源平台。

## 如何参与

我们欢迎大家通过 Pull Request 分享学习笔记、课程评价和考试回忆。详情请参阅我们的 [贡献指南](CONTRIBUTING.md)。
